# Kumo 节点详细界面

这里存放 Kumo 主题的节点详细界面截图，用于展示详情弹窗中的系统监控信息。

## 截图内容

- 节点基础状态与快速指标
- CPU、内存、网络速度、连接数、进程数图表
- 流量信息展示
- 系统、硬件和账单信息
- 网络延迟图表
